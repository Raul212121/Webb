# LuraMT2 Website

Website PHP pentru proiectul LuraMT2.

## Ce contine
- frontend clasic PHP
- pagini publice
- sistem login/register
- ranking players/guilds
- panou admin
- module donate, redeem, referrals, vote4coins

## Structura principala
- `index.php` - intrare principala
- `config.php` - configurare
- `pages/` - pagini site
- `include/classes/` - clase PHP
- `include/functions/` - functii si logica
- `include/db/` - fisiere de configurare si date
- `css/`, `js/`, `images/`, `img/` - resurse frontend

## Status
Proiectul este in analiza pentru reconstructie si curatare arhitecturala.
Scopul este refacerea siteului pe o baza mai clara, mai sigura si mai usor de extins.
